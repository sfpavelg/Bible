# Слой 1: Корень проекта

## Корневые файлы

- `README.md` - рабочая инструкция по запуску, сборке, публикации и git-процессу.
- `pubspec.yaml` - основной манифест Flutter-приложения: зависимости, версия, ассеты.
- `analysis_options.yaml` - правила статического анализа Dart/Flutter.
- `bible_app.iml` - файл IDE-модуля (IntelliJ/Android Studio).
- `.flutter-plugins` - список подключенных Flutter-плагинов (генерируется Flutter).
- `.flutter-plugins-dependencies` - карта зависимостей плагинов для платформ (генерируется Flutter).

## Корневые каталоги

- `lib/` - основной исходный код приложения.
- `assets/` - JSON-данные Библии и резервные копии.
- `android/` - Android-платформенная часть проекта.
- `ios/` - iOS-платформенная часть проекта.
- `web/` - web-конфигурация и web-ресурсы.
- `windows/` - Windows desktop-раннер и CMake-конфигурация.
- `.dart_tool/` - служебные артефакты Dart/Flutter (кэш, build-метаданные, web setup).

## Краткая карта слоев

1. Точка входа и UI находятся в `lib/`.
2. Текст Библии загружается из `assets/bible/`.
3. Для хранения заметок/журнала используется SQLite через `sqflite`.
4. Платформенные папки (`android`, `ios`, `web`, `windows`) обеспечивают сборку под конкретные устройства.
