# Слой 2: Исходный код (`lib`)

## Файлы верхнего уровня `lib`

- `lib/main.dart` - точка входа приложения; инициализирует `AppProvider`, настраивает `MaterialApp`, тему и размер шрифта.
- `lib/main_screen.dart` - главный контейнер с нижней навигацией между экранами: Библия, Блокнот, Журнал.

## Папка `lib/models`

- `lib/models/bible_model.dart` - доменные модели:
  - `BibleBook` (книги, сокращения, количество глав, принадлежность к Завету),
  - `BibleVerse` (представление стиха и сериализация в map).

## Папка `lib/services`

- `lib/services/bible_service.dart` - сервис загрузки/обработки текстов Библии из JSON:
  - загрузка Ветхого и Нового Завета из ассетов,
  - выборка книг, глав, стихов,
  - поиск по тексту,
  - служебные методы по сокращениям и числу глав.

## Папка `lib/database`

- `lib/database/database_helper.dart` - singleton-слой SQLite:
  - инициализация базы (включая web-вариант через `sqflite_common_ffi_web`),
  - создание таблиц `notes`, `reading_logs`, `bookmarks`, `reading_plans`,
  - CRUD-методы для заметок, журнала и закладок.

## Папка `lib/providers`

- `lib/providers/app_provider.dart` - состояние приложения (`ChangeNotifier`):
  - тема и размер шрифта,
  - текущие книга/глава,
  - флаг загрузки,
  - координация `BibleService` и `DatabaseHelper`,
  - публичные методы для UI (получение стихов, поиск, работа с заметками и журналом).

## Папка `lib/screens`

- `lib/screens/bible_screen.dart` - основной экран чтения:
  - переключение книг и глав,
  - свайпы по главам,
  - отрисовка стихов,
  - диалоги выбора книги/главы и поиска.
- `lib/screens/notebook_screen.dart` - экран блокнота (заготовка UI, TODO на добавление заметок).
- `lib/screens/journal_screen.dart` - экран журнала чтения (заготовка UI, TODO на календарь и добавление записей).
