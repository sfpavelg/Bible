# Документация по структуре проекта

Этот каталог содержит описание текущей структуры Flutter-проекта `bible_app` и назначения файлов по слоям.

## Список документов

- `docs/01_root_layer.md` - корневой слой проекта.
- `docs/02_lib_layer.md` - слой исходного кода Flutter (`lib`).
- `docs/03_assets_layer.md` - слой данных и ассетов (`assets`).
- `docs/04_android_layer.md` - Android-слой.
- `docs/05_ios_layer.md` - iOS-слой.
- `docs/06_web_layer.md` - Web-слой.
- `docs/07_windows_layer.md` - Windows-слой.
- `docs/08_generated_and_service_layer.md` - служебные и генерируемые файлы/папки.

## Принцип описания

- Документы отражают состояние проекта на текущий момент.
- Назначение указано для каждого файла в соответствующем слое.
- Для автогенерируемых каталогов с большим количеством файлов назначение описано по группам, чтобы документация оставалась поддерживаемой.
