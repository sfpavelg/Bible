## Полное описание

«Библия» — удобное приложение для ежедневного чтения Священного Писания на русском языке. Текст представлен в **Синодальном переводе**; при желании можно включить отображение элементов **Септуагинты** в квадратных скобках.

**Чтение**
- все книги Ветхого и Нового Завета;
- быстрый выбор книги и главы;
- листание глав жестом влево и вправо;
- настройка шрифта, размера, межстрочного интервала и интервала между стихами;
- светлая и тёмная тема.

**Поиск**
- поиск по слову или фразе по всей Библии;
- фильтры «Ветхий завет» / «Новый завет»;
- режим «Целое слово»;
- переход к найденному стиху с подсветкой.

**Избранное**
- сохранение выделенных стихов;
- копирование и управление записями.

**Блокнот**
- личные заметки и папки на устройстве;
- редактор с автосохранением;
- экспорт и обмен текстом.

**Планы чтения**
- параллельный, хронологический, последовательный и тематические планы («Вера», «Надежда», «Любовь»);
- план для начинающих;
- **«Стих для вдохновения»** — локальные напоминания в выбранное время, православный и протестантский календари, личные особые дни (без облака и без регистрации);
- отметки прогресса хранятся локально.

**Обновления**
- приложение может проверять наличие новой версии через интернет; установка обновления выполняется по вашему выбору.

Приложение **не требует регистрации**, **не показывает рекламу** и **не собирает аналитику**. Заметки, закладки и настройки хранятся на вашем устройстве.

## Декларация разрешений (блок «Безопасность данных»)

| Разрешение | Зачем | Передаётся на сервер? |
|------------|-------|------------------------|
| `INTERNET` | Проверка обновлений (JSON-манифест), экран техподдержки, открытие ссылок по действию пользователя | Нет персональных данных; только запрос публичного файла |
| `POST_NOTIFICATIONS` (Android 13+) | Локальные напоминания плана «Стих для вдохновения» — только после явного включения пользователем | Нет |
| `SCHEDULE_EXACT_ALARM` / `USE_EXACT_ALARM` | Точное время напоминания, заданное пользователем | Нет |
| Буфер обмена | Копирование стихов и вставка в блокнот по нажатию | Нет |
| WAKE_LOCK (если отображается) | Опция «Не выключать экран» при чтении | Нет |

**Не собираем:** геолокацию, контакты, камеру, микрофон, идентификаторы для рекламы, учётные записи.

## Сборка перед загрузкой

```bash
cd c:\Project\Bible\bible_app
# Создайте android/key.properties по образцу key.properties.example
flutter pub get
flutter build apk --release
```

APK: `build/app/outputs/flutter-apk/app-release.apk`

**Важно:** после смены `applicationId` установка поверх старого APK с `com.example.bible_app` **не обновит** приложение — это будет новая установка. Пользователям с Drive-версией сообщите о переходе на RuStore.

## FAQ для модерации (при необходимости)

**Почему нужен интернет?**  
Только для проверки обновлений и загрузки описания версии; основной текст Библии встроен в приложение.

**Есть ли монетизация?**  
Нет, приложение бесплатное.

**Где хранятся заметки?**  
Локально на устройстве пользователя.

**Есть ли push-уведомления?**  
Только локальные, в плане «Стих для вдохновения», если пользователь сам включил напоминания. Облачных push и FCM нет; текст Библии в уведомление не дублируется.
