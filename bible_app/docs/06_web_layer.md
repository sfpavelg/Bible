# Слой 6: Web (`web`)

## Файлы web-слоя

- `web/index.html` - HTML-шаблон web-приложения Flutter (точка загрузки JS рантайма).
- `web/manifest.json` - web app manifest (имя, иконки, режим запуска PWA).
- `web/sqflite_sw.js` - service worker/скрипт поддержки `sqflite_common_ffi_web` для работы web-базы.
