# Архив: подготовка JSON-текстов Библии

Скрипты и исходники этапа **верификации и правки** текстов до переноса в приложение.

**Не используются** при сборке релиза. Актуальные файлы для приложения:

- `bible_app/assets/bible/new_testament_correct.json`
- `bible_app/assets/bible/old_testament_correct.json`

## Содержимое

| Папка | Что внутри |
|-------|------------|
| `scripts/` | Python-скрипты `fix_*`, `validate_*`, `check_*` |
| `data/` | `correct_bible.json`, `russian_synodal_bible.json` — эталоны для сверки |
| `reports/` | Отчёты валидации ВЗ и НЗ |

## Повторный запуск

Пути в скриптах указывали на корень репозитория и `bible_app/assets/…` — при необходимости поправьте вручную или запускайте из корня с обновлёнными путями. Архив сохранён для истории и редких правок текста.
